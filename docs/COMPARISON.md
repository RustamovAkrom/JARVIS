# Сравнение с другими ассистентами

Alexa:
Работает только через облако.
Требует аккаунты.
Нет локального контроля.

Google Assistant:
Зависит от Google сервисов.
Нет офлайн режима.

Siri:
Только экосистема Apple.

Jarvis:

- Работает локально
- Не требует аккаунтов
- Поддерживает офлайн
- Открытый код
- Полный контроль пользователем
- Можно кастомизировать

Jarvis — это ассистент для разработчиков и power users.
