# Система команд

Jarvis поддерживает:

- Нечёткое сравнение (fuzzy matching)
- Несколько команд в одном предложении
- Разговорные фразы
- Разные формулировки одной команды

Пример:

"Открой браузер и потом включи ютуб"

Jarvis разобьёт фразу и выполнит команды по очереди.

Команды работают быстрее AI и всегда имеют приоритет.
