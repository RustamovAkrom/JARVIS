# AI режим

AI режим использует Gemini API.

Особенности:

- Работает только при наличии интернета
- Можно отключить через config.yaml
- Не мешает системным командам
- Использует оптимизированный voice prompt

Jarvis автоматически переключается между:

System Mode
AI Mode

Это позволяет избежать конфликтов.
