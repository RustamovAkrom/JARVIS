# Jarvis Documentation

Добро пожаловать в документацию Jarvis Voice Assistant.

Jarvis — это локальный голосовой ассистент для Windows с возможностью офлайн работы и интеграцией AI.

Основные разделы:

- INSTALLATION.md — установка
- CONFIGURATION.md — настройка
- ARCHITECTURE.md — устройство системы
- VOICE_SYSTEM.md — голосовой пайплайн
- AI_MODE.md — AI режим
- COMMAND_SYSTEM.md — система команд
- FEATURES.md — возможности
- COMPARISON.md — сравнение с другими ассистентами
- ROADMAP.md — планы развития
